What is Metro 2.0?

Metro 2.0 is a website, which is going to be used widely for downloading the desired icons and for customizing it. Also we have planned to export this as a standalone plug-in for using it along with Syncfusion EJ Web controls. To be more clear, we will export all the icons as SVG path in a JS file. The user refer this file in html page and can use the desired icons.

How does this differ from other icon editors?

The Metro 2.0 is developed as Web Application using ionic and angular it can be run widely in both deskto and mobile devices. Both android and windows phones supports metro 2.0.It provides extra feature of downloading the desired icons as separate widget and downloading them .

What are the advantages over Metro Studio?

The Metro 2.0 is Web Application and it supports in all the mobile devices.The Metro 2.0 provides options for creating the images and loading the icons and downloading them The Metro 2.0 provides the creation of custom widgets and downloading them as the packages

What are the modules present it it?

The Metro 2.0 consists of the below modules
  • Icon collections
  • Editing the icons
  • Custom Editor
  • Download plugin

What are the formats supported for download?

The formats supported as JPEG,PNG,PDF and SVG formats

How to use it as a library?

When the button is clicked the icons are downloaded as package.The icons can be customized and then downloaded. The Downloaded icons can also be exported to the required formats.
